# 🚀 QUICK REFERENCE: Build APK in 5 Commands

## Copy-Paste These Commands:

```bash
# 1. Install EAS CLI (global, one-time)
npm install -g eas-cli

# 2. Navigate to project
cd /app/react-native-ar-survey

# 3. Login to Expo (you'll need to create free account at expo.dev)
eas login

# 4. Install dependencies
npm install

# 5. Build APK
eas build --platform android --profile preview
```
